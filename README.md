# Dev Playground

Repositorio personal para guardar proyectos de práctica, experimentos y ejercicios de aprendizaje.

## Propósito
Centralizar proyectos pequeños y medianos hechos mientras practico distintas herramientas del ecosistema frontend y mobile, para no perderlos y seguir ampliándolos con el tiempo.

## Proyectos
- `business-website-nextjs-tailwind`

## Stack que puede aparecer en este repositorio
- React
- Next.js
- Tailwind CSS
- TypeScript
- Expo
- React Native

## Observación
La mayoría de los proyectos aquí son de práctica, cursos, pruebas técnicas o ejercicios de aprendizaje.
